# News Checklist

**First, determine the story type:**
- **Original** — A PBS Wisconsin News production
- **Syndicated** — From AP, WPR.org, or another outside news partner

---

## Original Story

- [ ] Asset check
  - [ ] Video
  - [ ] Captions
  - [ ] Transcript
- [ ] Copy doc with producer:
  - [ ] Initial script (copy into new doc to edit into article format. Wait until producer confirms the script reflects the final edit)
  - [ ] (for packages and clips) Media Manager copy
  - [ ] (for packages) Article edit of package copy
  - [ ] Edits approved
- [ ] Post video to Media Manager
- [ ] Post News item
- [ ] Slack alert to team
- [ ] Post to Facebook
- [ ] Post to Facebook again
- [ ] Check for comments

---

## Syndicated Story

- [ ] Original
- [ ] Image (Video Search if possible)
- [ ] Edit for house style
- [ ] Write 200-300 single sentence short description
- [ ] Post
- [ ] Check for story updates throughout day
- [ ] Slack alert to team
- [ ] Post to Facebook
- [ ] Check for comments
