# Essentials

Use this list as the basic checklist for ANY PBSWI content to be posted.

Content for broadcast should be posted two weeks prior to premiere at minimum for automation/scheduling purposes.

## File Check

- [ ] 1080p video
- [ ] Caption files
- [ ] Pull stills — ideally three
- [ ] Post stills to AirTable
- [ ] Check for clips/digital shorts — clips should have SSoT entries, digital shorts can function as content calendar tasks

## Confirm platforms

(add content calendar entry for each platform, with associated checklist)

- [ ] Media Manager
- [ ] YouTube
- [ ] sIX
- [ ] PBS LearningMedia
- [ ] Website — PBSWisconsin.org custom landing page
- [ ] EDU Website — PBSWisconsinEducation.org custom landing page

## SSoT

- [ ] Confirm digital release date
- [ ] Initial draft
- [ ] Digital copy pass — SEO, headline rework, tags
- [ ] PR copy edit
- [ ] Confirm thumbnails with art director/producer
- [ ] Copy approved for use

## Data check

- [ ] IMDb entry
- [ ] EiDR entry
- [ ] Episode
- [ ] Edit (needed for sIX posting)
- [ ] Manifestation (needed for sIX posting)
- [ ] Confirm on schedule page

## Producer checkpoint

- [ ] Add embed, PBSWI links and YouTube links to SSoT
- [ ] Post embed and links in Slack
