- [ ] Copy/paste into AirTable record

- [ ] Post

## Create Content Project

Think of this as a series bucket on Media Manager.

- [ ] Check if one already exists
- [ ] Outline objects and assets
- [ ] Create as needed

## Create Objects

Think of as individual uploads of video files and caption files.

Create objects for assets —

### Required Metadata

- [ ] Title -> Animated Video | TITLE | Wisconsin Biographies
- [ ] Video source file -> MP4
- [ ] Poster image -> Slate
- [ ] Geo Restrictions -> Worldwide
- [ ] Related Objects -> Main version
- [ ] Caption file -> VTT only

## Create Assets

Containers for Videos, Media, Interactives, HTMLFragments and images that include Metadata, Credits, Rights. Think of this as the clip/full asset level on Media Manager.

**WARNING:** you can't save an Asset without associating it with an Object first, so be sure that you have created all the necessary objects first.

Create assets for each object —

### Required Metadata

- [ ] Content Project -> Current collection or season
- [ ] Title -> Meet X, take a look, etc.
- [ ] Related objects -> Video, language versions. Set as primary asset, with additional language options marked as needed.
- [ ] Poster image -> 16x9. Use Screenshots for Assets other than main video
- [ ] Description -> Card Description
- [ ] Asset type -> Animation, Video Episode, Video Exercise, etc.
- [ ] Media Type General -> Video
- [ ] Audience -> BE MINDFUL HERE. For Educators/Teachers if explanatory, For Use with Students if part of the lesson.
- [ ] Credits -> Copy from landing page
- [ ] Ownership and Rights ->
  - © 2025 Wisconsin Educational Communications Board and The Board of Regents of the University of Wisconsin System
  - Stream, Download and Share
  - Never Expires
- [ ] Accessibility Indicators -> Caption, Transcript
