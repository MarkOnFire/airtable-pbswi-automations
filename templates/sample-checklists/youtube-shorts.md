# YouTube Shorts Publishing Checklist

## Video Preparation

- [ ] Video is under 60 seconds
- [ ] Vertical format (9:16 aspect ratio)
- [ ] Resolution: 1080x1920 or higher
- [ ] Engaging hook in first 3 seconds
- [ ] No letterboxing or black bars

## Content Requirements

- [ ] Fast-paced, engaging content
- [ ] Clear audio (music or narration)
- [ ] Text overlays readable on mobile
- [ ] PBS Wisconsin branding (subtle)
- [ ] Captions/subtitles burned in (recommended)

## Upload & Metadata

- [ ] Video uploaded as Short (vertical, under 60s)
- [ ] Title set (include #Shorts if needed)
  - [ ] Keep title concise and catchy
  - [ ] Include relevant keywords
- [ ] Description added:
  - [ ] Brief content summary
  - [ ] Link to full video (if applicable)
  - [ ] PBS Wisconsin links

## Settings

- [ ] Visibility set (Public/Scheduled)
- [ ] Scheduled for {premiere_date}
- [ ] Comments enabled
- [ ] Audience set (made for kids: No)

## Optimization

- [ ] Trending sounds/music used (if applicable)
- [ ] Relevant hashtags in description
- [ ] Connected to main channel content

## Post-Publish

- [ ] Verify Short is live and playing
- [ ] Check it appears in Shorts feed
- [ ] Cross-promote on other platforms
- [ ] Monitor performance metrics
- [ ] Respond to comments

## Cross-Promotion

- [ ] Share to Instagram Reels (adapted version)
- [ ] Share to TikTok (if applicable)
- [ ] Link from full-length video
