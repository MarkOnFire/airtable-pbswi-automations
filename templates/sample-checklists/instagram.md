# Instagram Publishing Checklist

**Project:** {project_name}
**Premiere Date:** {premiere_date}

---

## Content Preparation

- [ ] Image(s) selected and edited
- [ ] Video clip prepared (under 60 seconds for feed, 90 for Reels)
- [ ] Caption drafted
- [ ] Hashtag research completed
- [ ] Alt text written for accessibility

## Visual Requirements

- [ ] Image/video meets Instagram specs:
  - [ ] Square (1:1) or Portrait (4:5) aspect ratio
  - [ ] Minimum 1080px width
  - [ ] High quality, not pixelated
- [ ] PBS Wisconsin branding visible (if applicable)
- [ ] Text on image is readable

## Caption & Hashtags

- [ ] Caption includes:
  - [ ] Engaging hook/first line
  - [ ] Content description
  - [ ] Call to action
  - [ ] Credit tags (photographer, talent)
- [ ] Hashtags prepared (15-30 relevant tags)
- [ ] Hashtags include:
  - [ ] #pbswisconsin
  - [ ] Content-specific tags
  - [ ] Location tags

## Posting Setup

- [ ] Correct Instagram account selected
- [ ] Post type selected (Feed, Reel, Carousel)
- [ ] Location tagged
- [ ] People tagged (with permission)
- [ ] Alt text added

## Scheduling

- [ ] Post scheduled for optimal time on {premiere_date}
- [ ] Story post prepared (if applicable)
- [ ] Cross-post to Facebook enabled (if applicable)

## Stories (if applicable)

- [ ] Story content created
- [ ] Link sticker added (if eligible)
- [ ] Mention stickers for partners
- [ ] Poll/question stickers (for engagement)

## Post-Publish

- [ ] Verify post went live
- [ ] Add to relevant Highlights (if applicable)
- [ ] Respond to early comments
- [ ] Share to Story
- [ ] Monitor engagement
