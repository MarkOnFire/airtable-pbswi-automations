# PBS Media Manager Publishing Checklist

**Project:** {project_name}
**Premiere Date:** {premiere_date}

---

## Pre-Ingest Preparation

- [ ] Video file meets PBS specifications
- [ ] Closed captions file (SRT/VTT) ready
- [ ] Thumbnail image prepared (16:9)
- [ ] All metadata gathered

## Media Manager Entry

- [ ] Logged into Media Manager
- [ ] Correct show/series selected
- [ ] New episode/asset created

## Required Metadata

- [ ] Title (official release title)
- [ ] Description/synopsis
- [ ] Episode number (if applicable)
- [ ] Season number (if applicable)
- [ ] Duration
- [ ] Original air date: {premiere_date}

## Content Classification

- [ ] Content type selected (Full Episode, Clip, etc.)
- [ ] Genre/category assigned
- [ ] Audience rating set
- [ ] Content advisories added (if applicable)

## Rights & Availability

- [ ] Rights holder information entered
- [ ] Availability window configured:
  - [ ] Start date: {premiere_date}
  - [ ] End date set
- [ ] Geo-restrictions configured
- [ ] Passport availability set (if applicable)

## File Upload

- [ ] Video file uploaded/ingested
- [ ] Transcoding completed successfully
- [ ] Captions uploaded and synced
- [ ] Thumbnail uploaded

## Quality Verification

- [ ] Preview playback in Media Manager
- [ ] Captions sync correctly
- [ ] Audio levels appropriate
- [ ] No encoding artifacts

## Publishing

- [ ] Status set to "Publish"
- [ ] Published to:
  - [ ] PBS.org (national)
  - [ ] pbswisconsin.org (local)
  - [ ] PBS App
- [ ] Verify content appears on destination platforms

## Documentation

- [ ] Media Manager ID recorded
- [ ] Link back to SSOT record
- [ ] Team notified of publication
