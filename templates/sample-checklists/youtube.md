# YouTube Publishing Checklist

**Project:** {project_name}
**Premiere Date:** {premiere_date}

---

## Pre-Upload

- [ ] Video file exported (1080p MP4, H.264)
- [ ] Closed captions (SRT) prepared and reviewed
- [ ] Thumbnail created (1280x720, PNG or JPG)
- [ ] Video description drafted with PBS Wisconsin branding

## Upload & Metadata

- [ ] Video uploaded to YouTube
- [ ] Title set: "{project_name}"
- [ ] Description added with:
  - [ ] Episode/content summary
  - [ ] PBS Wisconsin branding
  - [ ] Relevant links (website, social)
  - [ ] Credits/acknowledgments
- [ ] Tags added (minimum 5 relevant tags)
- [ ] Category selected (typically "Entertainment" or "Education")
- [ ] Closed captions uploaded and synced

## Publishing Settings

- [ ] Premiere scheduled for {premiere_date}
- [ ] Visibility set to "Scheduled" or "Public"
- [ ] End screen configured (subscribe button, related videos)
- [ ] Cards added at relevant timestamps
- [ ] "Notify subscribers" enabled
- [ ] Comments enabled/moderated as appropriate

## Thumbnail & Branding

- [ ] Custom thumbnail uploaded
- [ ] Thumbnail includes PBS Wisconsin branding
- [ ] Thumbnail is readable at small sizes

## Post-Publish

- [ ] Added to appropriate playlist(s)
- [ ] Pinned comment added (if applicable)
- [ ] Social media cross-post scheduled
- [ ] Shared link with team for promotion
- [ ] Verify playback and captions working
