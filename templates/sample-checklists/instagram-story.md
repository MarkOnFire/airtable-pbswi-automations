# Instagram Story Publishing Checklist

**Project:** {project_name}
**Premiere Date:** {premiere_date}

---

## Content Preparation

- [ ] Story content created (image or video)
- [ ] Vertical format (9:16 aspect ratio)
- [ ] Video under 15 seconds per slide
- [ ] Multiple slides planned (if applicable)
- [ ] Visual hierarchy clear on mobile

## Design Requirements

- [ ] Safe zones respected (no text in top/bottom 250px)
- [ ] Text is large and readable
- [ ] Contrast is sufficient
- [ ] PBS Wisconsin branding included
- [ ] Consistent visual style

## Interactive Elements

- [ ] Link sticker added (to full content)
- [ ] Poll sticker (for engagement)
- [ ] Question sticker (for interaction)
- [ ] Quiz sticker (if educational)
- [ ] Countdown sticker (for premiere)
- [ ] Location sticker
- [ ] Mention stickers (@tags)

## Story Sequence (if multi-slide)

- [ ] Slide 1: Hook/attention grabber
- [ ] Slide 2-3: Main content
- [ ] Final slide: Call to action

## Scheduling

- [ ] Stories scheduled for {premiere_date}
- [ ] Timing aligned with feed post
- [ ] Duration appropriate (not too many slides)

## Cross-Promotion

- [ ] Feed post linked/referenced
- [ ] "New Post" sticker used (if applicable)
- [ ] Swipe-up/link to YouTube or website
- [ ] Partner accounts tagged

## Post-Publish

- [ ] Verify all slides posted correctly
- [ ] Check interactive elements work
- [ ] Monitor replies/reactions
- [ ] Respond to DMs and replies
- [ ] Add to Highlight (if evergreen content)

## Highlight Consideration

- [ ] Content suitable for Highlight?
- [ ] Highlight cover designed
- [ ] Highlight category identified
