##  Overview
The following images and logos are used to represent shows across PBS web, mobile and OTT platforms.
Please follow the guidelines to ensure that your images display correctly.
Image | Details | Example (click images to expand)  
---|---|---  
Image | Details | Example (click images to expand)  
---|---|---  
Show Logo 4:1 | 
  * Dimensions: 680 pixels wide x 170 pixels high
  * File type: PNG
  * Aspect ratio: 4:1
  * Alignment: Left-aligned horizontally, center-aligned vertically
  * Upload white (White Logo 4:1) and color (Color Logo 4:1) logos

[ Click here to download the 4:1 PSD Template with guidelines and examples ](https://docs.pbs.org/download/attachments/5343808/4x1-Show-Logo-Template%20\(3\).psd "Click here to download the 4:1 PSD Template with guidelines and examples") [See details](https://docs.pbs.org/space/MM/5343864/4%3A1+Show+Logos) |  ![Show-Logo-4x1.png](https://docs.pbs.org/download/attachments/5343808/Show-Logo-4x1.png?version=1&modificationDate=1731945823362&cacheVersion=1&api=v2)  
Show - Poster 2x3 | 
  * Dimensions: 2000 pixels wide x 3000 pixels high
  * File type: JPG
  * Aspect ratio: 2:3
  * File size max: 2.5MB

[See details](https://docs.pbs.org/space/MM/5345629/Show+-+Poster+2%3A3) |  ![](https://docs.pbs.org/download/attachments/5343808/show-poster-example-2020.jpg?version=1&modificationDate=1580497525612&cacheVersion=1&api=v2)  
Background | 
  * Dimensions:
    * Minimum: 2880 pixels wide x1620 pixels high
    * Maximum: 3840 pixels wide x 2160 pixels high
  * File type: PNG, JPG
  * Aspect ratio: 16:9
  * File size max: 2.5MB

This image is used for the following:
  * Background for featured Passport content
  * The social share image when shows are shared on social media
  * Fallback image when no Featured Preview Content ID is available on Show Detail screens

This image profile is used across all Digital products so it’s recommended that all shows have a background image uploaded.
